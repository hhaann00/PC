## PC

**요약**

- 2020-1 , 2학기 Java 프로그래밍 과목을 수강한 후 이를 복습하고자 eclipse와 MySQL을 연동해 간단한 PC방 프로그램을 개발하였습니다.

**역할**

- 개발자 3명으로 구성
- 역할을 프론트엔드 / 백엔드로 나누지 않고 기능별로 시스템 파트를 나눠 모든 부분 개발에 참여하였습니다.
- 맡은 역할 : 회원관리 부분 / 1:1 채팅 / 데이터베이스 기획 & 생성

**성과**

- 팀 프로젝트를 통한 협업 능력 향상
- 데이터베이스 기획 및 구성 능력 향상
- 데이터베이스에 대한 이해도 향상
- Java 클래스 활용 능력 향상
- SQL문 사용 능력 향상.
